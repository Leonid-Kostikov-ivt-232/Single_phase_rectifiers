# Single_phase_rectifiers

Программа предназначена для расчета схем однофазных выпрямителей с емкостным сглаживающим фильтром по заданным пользовательским данным.

## Технологии

- **IDE**: Code::Blocks — бесплатная кроссплатформенная среда разработки для C/C++ проектов.
- **Язык**: C++.

## Установка и запуск

1. Установите Code::Blocks с GNU GCC Compiler.
2. Откройте проект в IDE: `File → Open → [имя_проекта].cbp`.
3. Соберите проект: `Build → Build`.
4. Запустите: `Build → Run`.
